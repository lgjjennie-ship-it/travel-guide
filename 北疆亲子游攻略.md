## 📅 10天详细行程

### Day 1｜各地 → 乌鲁木齐
**主题**：抵达休整

| 时间 | 安排 |
|------|------|
| 全天 | 抵达乌鲁木齐，入住酒店 |
| 傍晚 | 国际大巴扎逛逛，吃烤肉、买干果 |
| 晚餐 | 大盘鸡、烤羊肉串、烤包子、酸奶 |

**住宿**：乌鲁木齐市中心  
**🧒 亲子Tips**：第一天别安排太满，调整时差和状态

---

### Day 2｜乌鲁木齐 → 赛里木湖
**车程**：约 550km / 4-5h（G30高速）

| 时间 | 安排 |
|------|------|
| 08:30 | 酒店早餐 |
| 09:00 | 出发，走 G30 连霍高速 |
| 12:00 | 途中午餐（奎屯/精河服务区） |
| 13:30 | 经过**果子沟大桥**（壮观！停车拍照） |
| 14:30 | 抵达赛里木湖，入住湖边 |
| 17:00 | 湖边散步、玩耍 |
| 21:00 | 看日落（新疆日落晚） |

**住宿**：赛里木湖边民宿/酒店  
**🧒 亲子Tips**：果子沟大桥小朋友会很兴奋；湖边可以跑、放风筝、捡石头

---

### Day 3｜赛里木湖（深度游）
**主题**：环湖慢游 + 发呆放空

| 时间 | 安排 |
|------|------|
| 睡到自然醒 | 湖边早餐 |
| 10:00 | 开始环湖自驾（约70km，边走边停） |
| 12:00 | 找个喜欢的湖湾野餐 |
| 下午 | 湖边发呆、拍照、玩水 |
| 21:00 | 看日落 |
| 晚上 | 如果天气好，可以看星空 |

**住宿**：赛里木湖（可换蒙古包体验）  
**🧒 亲子Tips**：带野餐垫+零食；6月湖水还凉，可以踩水但别下去游

---

### Day 4｜赛里木湖 → 那拉提草原
**车程**：约 300km / 4h

| 时间 | 安排 |
|------|------|
| 09:00 | 出发，走 G218 |
| 11:00 | 途经霍城，可看**薰衣草**（6月初开始开花） |
| 13:00 | 抵达那拉提镇，午餐 |
| 14:00 | 入住酒店/民宿 |
| 下午 | 镇上闲逛，感受哈萨克风情 |
| 晚餐 | 手抓肉、奶茶、馕 |

**住宿**：那拉提镇/景区周边  
**🧒 亲子Tips**：薰衣草拍照很出片；镇上有卖哈萨克小帽子、马鞭等纪念品

---

### Day 5｜那拉提草原（深度游）
**主题**：空中草原 + 骑马 🐴

| 时间 | 安排 |
|------|------|
| 09:00 | 进入那拉提景区（门票+区间车） |
| 10:00 | 坐区间车到**空中草原** |
| 11:00 | 草原徒步、拍照 |
| 12:30 | 草原野餐 / 牧民家午餐 |
| 14:00 | **骑马体验**（重头戏！） |
| 17:00 | 返回镇上 |
| 21:00 | 草原日落 |

**住宿**：那拉提  
**🧒 亲子Tips**：骑马可以大人和孩子共骑；让马夫牵着走，安全第一；6月草原野花盛开

---

### Day 6｜那拉提 → 巴音布鲁克
**车程**：约 150km / 3h

| 时间 | 安排 |
|------|------|
| 睡懒觉 | 不着急出发 |
| 11:00 | 出发，翻越天山 |
| 途中 | 风景绝美！雪山+草原，走走停停拍照 |
| 14:00 | 抵达巴音布鲁克镇，午餐+入住 |
| 16:00 | 休息一下 |
| 18:00 | 进入景区，坐区间车去**九曲十八弯** |
| 21:00 | 看**九曲十八弯日落**（此行最震撼！） |

**住宿**：巴音布鲁克镇  
**⚠️ 重要**：九曲十八弯日落是整个行程的高光！提前2小时进景区；海拔2500m，带厚外套

---

### Day 7｜巴音布鲁克 → 独库公路 → 乌鲁木齐
**车程**：约 500km / 6-7h（唯一长途日）

| 时间 | 安排 |
|------|------|
| 07:30 | 早起出发（今天路程长） |
| 08:00 | 走**独库公路北段**（中国最美公路！） |
| 途中 | 雪山→峡谷→草原→森林，一路变换 |
| 12:00 | 乔尔玛午餐 |
| 12:30 | **乔尔玛烈士陵园**（可停留，给孩子讲修路故事） |
| 下午 | 各观景台停车拍照 |
| 18:00 | 抵达乌鲁木齐 |

**住宿**：乌鲁木齐  
**⚠️ 注意**：独库公路一般**6月1日**开通，出发前确认路况  
**🧒 亲子Tips**：准备iPad+动画片+零食；独库公路弯道多，晕车提前吃药

---

### Day 8｜乌鲁木齐（休整日）
**主题**：躺平恢复 😴

| 时间 | 安排 |
|------|------|
| 全天 | 睡懒觉！酒店发呆 |
| 下午 | （可选）**新疆博物馆**（免费，看楼兰美女） |
| 晚餐 | 吃顿好的 |

**住宿**：乌鲁木齐  
**🧒 亲子Tips**：带娃旅行，中间安排休整日很重要！

---

### Day 9｜天山天池 / 市区休闲
**主题**：轻松一日游

#### 选项A：天山天池（车程1.5h）
- 09:00 出发去天池
- 10:30 抵达景区，坐大巴上山
- 11:30 天池湖边漫步
- 14:00 坐游船
- 16:00 下山返回市区

#### 选项B：市区休闲
- 上午：红山公园（俯瞰乌鲁木齐）
- 下午：大巴扎购物，采购手信

---

### Day 10｜乌鲁木齐 → 返程

| 时间 | 安排 |
|------|------|
| 上午 | 睡懒觉，收拾行李 |
| 中午 | 最后一顿新疆美食 |
| 下午 | 前往机场，飞回家 ✈️ |

## ⚠️ 注意事项

1. **时差**：新疆日出约7:00，日落约21:00-21:30
