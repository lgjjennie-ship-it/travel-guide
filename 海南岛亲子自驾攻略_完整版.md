### 🏛️ 骑楼老街

- **门票**：免费
- **停车**：建议打车去，淡季也是人从众，开车不方便
- **我觉得还不错的食物**：海南粉、清补凉、老盐柠檬水（有连锁店）
- **博物馆**：骑楼老街博物馆（免费，了解华侨下南洋历史）

### 🏖️ 假日海滩

- **门票**：免费
- **日落时间**：约18:45-19:15，看每日日落时间
- **提示**：海口就简单逛逛，中转站

## 📍 Day 2～3：海口→文昌→文昌深度游

### ⏰ 时间线

| 行程 | 美食推荐 | 备注 |
|------|------|------|
| 出发去文昌 | 买几杯老盐柠檬水带走，开车解乏 | 走**滨海路**（不要走高速，沿途看海景椰林和跨海大桥） |
| 🌿 东寨港红树林（坐船30min） | / | 免费，作为备选景点，时间充裕可以去看看，湿地生态系统或教材，可以看成群小螃蟹 |
| 🏛️ 铺前老街（会经过跨海大桥） | / | 比较原生态，当地人自己生活的镇，有一些骑楼，但目前毁的差不多，也有很多渔村，需要开车弯过去 |
| 木兰湾 | / | 亚洲第一灯塔，文昌最北端的原生态海湾，因为没有过度开发，所以沙粒和其他湾区相比体验差，可是更原生态，这边海浪很大 |
| 抱虎角 | / | 途径的一个沙滩，原生态果冻海，人极少，沙子也不错，也有远处的栈道，拍照挺好看 |
| 月亮湾 | ⭐⭐ | 人很少，沙子质量高，很细腻，水清澈，这边看日落很美，他方向朝西 |
| 铜鼓岭景区+石头公园 | / | 付费的景点，可以爬上去看日出/日落，可以俯瞰整个月亮湾，值不值得看个人想法。石头公园是火山岩海岸，有各种各样的石头，小孩子可以探险。 |
| 🌙 月亮湾看日落 | ⭐⭐ | 月亮湾方向朝西，可以看日落，也可以去铜鼓岭山顶看日落 |
| 文昌航天科普中心 | / | 说实话，我觉得和江浙沪的航天展相比，差的不是一点半点，门票还贵，已开放的几个房间质量不高，不如上海天文中心，但远道而来航天迷可以去看看。那一圈都是商发区域，可以看到商发塔台，提前关注公众号可以了解火箭发射计划，报名参观。商发中心还在建设中。 |
| 淇水湾海滩 | / | 龙楼镇附近比较不错的沙滩，也比较出名 |
| 🌴 东郊椰林 | / | 是比较热门的景点，但了解过文昌历史就知道没必要专门去这里，文昌椰子产量占了海南岛50%+，整个沿海线路和内部的村镇路上都是椰子林，夕阳下拍照都很美 。东郊椰林景区范围非常大，路边宰人的事情也比较多，需要自己判断|
| 石鼻岛赶海 | ⭐⭐⭐⭐ | 个人强推的地点，它在当地居民村庄后面，要穿过椰林，很少有游客过去，没啥开发，红树林生态滩涂岛,海边都是冲上来的珊瑚礁，非常清净，浪小，海水颜色也很有层次，还有几颗快灭绝的红树林,可同时看白鹭、观鸟，适合亲子上午去赶海。中午去还是挺晒的。|
| 牛奶海 | / | 琼海市潭门镇孟菜园村（环岛旅游公路旁），比较热门的景点，海水呈乳白色/奶蓝色，视觉效果独特，海滩由白色珊瑚石构成.其实他就在石鼻岛对面，但中间隔着石头带|
| 任意门 | / | 比较热门的网红拍照景点，大爷大妈们更喜欢去😊，有一些无人机商拍，这块的沙滩和海和其他地区比起来差点意思，有点儿像看长江的感觉。它的特色：这里是当地渔村的停泊点，可以看到海量的渔船，还能看到渔船出海的仪式感，这里也能看到不断巡航的海警|

### 🍗 文昌鸡（网友说必吃，可是我们都不喜欢，太肥了，不如上海三黄鸡）
- **避雷**：铺前老街上的三家别吃了，文昌鸡肥到生理性厌恶，感觉都是速成鸡。
- **价格**：基本上¥50/斤

### 🏠住宿推荐
- 文昌龙楼镇有一家"文昌福楼朗岳酒店(文昌龙楼航天城店)"，体验下来性价比都不错，装修、服务都还行，但如果遇到火箭发射日，价格会很高。

### 🏨吃饭推荐
- 龙楼镇上几家海南粉、抱罗粉都不错，当晚饭和早饭都可。
- 街上当地人开的清补凉算是整个行程中味道最好、食材最正宗、价格最便宜的。越往后走，清补凉味道越来越糖精味。
- 街上买水果和零食都比较方便，适合自驾补给。

## 📍 Day 4：文昌→琼海→万宁的石梅湾

| 行程 | 简介 |
|------|------|
|出发去万宁|/|
| 潭门中心渔港 | 早上去感受渔港盛况，也可以买海鲜去找店铺加工 |
| 玉带滩 | 要门票，一条自然形成的狭长沙洲，把万泉河和南海分隔开来——左边是万泉河，右边是南海，站在沙滩上能同时看到"河"和"海。2005年吉尼斯认证"最狭窄分隔海河的沙滩半岛"，可以玩个1小时，但这边没有航拍的话，没那么深的感受 |
| 南海博物馆、博鳌亚洲论坛永久地 | 玉带滩在琼海的博鳌论坛附近，南海博物馆就在旁边，可以给看海行程增加一些不同的文化底蕴。26年的博鳌亚洲论坛也在那边举行 |
| 燕子洞 & 正门岭大桥 ⭐️⭐️⭐️⭐️ | （天气好的情况下）个人觉得是最美最干净最有气魄的沙滩。亿万年海浪侵蚀形成的天然海蚀洞，沙粒大，所以海水不浑浊；周边黑礁石嶙峋，惊涛拍岸极具视觉冲击；一面是山，一面是海，画面感比较强。注意不要被路边开沙滩车的人骗了，也不会走几步路就到燕子洞了，涨潮时海浪穿洞，很震撼 |
| 🌿 兴隆热带植物园 | **门票**：成人60元，儿童30元，归侨文化精髓。但服务跟不上票价，可作为备选地，绿植和拍照爱好者可以看看 |
| 神州半岛 | 沙滩和水质比较好，比较热门的沙滩 |
| 桥海牧场 | 文昌去往万宁，我们一直走的沿海公路，所以一路上会经过很多当地村庄，随处可见各种菠萝地、凤梨地、西瓜地、牧场、漫山遍野的芒果树，比较出名的是桥海牧场，赏心悦目|
| 住宿 | 住宿可以选择神州半岛、石梅湾、日月湾这3个湾区，神州半岛中档酒店多，但酒店评价里基本会反馈房间潮湿/蚊虫多/卫生不过关/设备老旧/皮肤过敏等问题，住这里订酒店需要花时间筛选。石梅湾开发成熟，高档酒店多，基本自带沙滩，但人也多，订购酒店基本要在OTA平台套餐价基础上再额外加价（我看的时候3月底、4月初房价基本要每晚额外再加七八百）。日月湾，冲浪圣地，运动气息强烈些，距离石梅湾十几分钟路程，国家冲浪训练基地 |

## 📍 Day 5：石梅湾/日月湾(躺平日)

| 行程 | 简介 |
|------|------|
| 🌲 青皮林徒步 | （如前一天没走） |
| 🏄 日月湾观景/冲浪 | 我们住在"万宁日月湾中旅逐浪樾潮酒店"，冲浪主题的酒店，订购时候可以选择覆盖冲浪的房间，可冲浪90min（30min教学+60min冲浪池训练），有教练教学+免费提供设备。酒店服务很好、绿化设计很赞、有恒温泳池，早饭也不错，适合不同需求的一家人躺平一天。也可以在日月湾看别人冲浪。不足点在于，酒店周边正在造路，进出酒店的路有1公里不顺畅，晚上开车没有路灯。 如果住在这里，可以和前台协商能不能免费升级到高楼层（入住客人少的情况下），一般OTA订购的冲浪房型在1～3楼，看不到海。我们去的时候人流量适中，协商后免费升级到高楼层海景房了，可以看到远处的海天一线、早上也能看到半路子的日出、也能看到周边的配套设施。这个酒店开车2～3min后，到日月湾商业街，有一家吃海南鸭肉的连锁店（排队人最多、店名我忘了），口味还行，有团购套餐399，适合一家三口|

## 📍 Day 6：石梅湾→清水湾

### ⏰ 时间线

| 行程 | 简介 |
|------|------|
| 前往清水湾 | 继续走滨海旅游公路 |
| 清水湾 | 沿海公路途经点 |
| 抵达清水湾，入住 | - |
| 南湾猴岛+新村港疍家渔排 | 野生猕猴小岛。小红书攻略都是推荐买缆车+景区门票(¥163)，其实景区的猴子满岛窜，可以坐旁边私人开的水上公交，往返¥18，进去也能自己喂猴子。水上公交会在疍家渔排中间穿过，疍家人是"水上吉普赛人"，老一辈渔民还住在上面。 |
| 岭仔村赶海点 ⭐⭐⭐ | 陵水赶海的亲子活动比较多，可以看自己需求找一下，比如岭仔村、比如陵水红树林潮汐营地等～～ |
| 🏖️ 清水湾-会唱歌的沙滩 | 世界三大会唱歌的沙滩之一，沙子踩上去"咯吱咯吱"响（石英含量高 98%+），上午或傍晚的时候可以去玩一趟 |

### 🍜 陵水酸粉（必吃！）

- **推荐**：银河酸粉（陵水县城）
- **价格**：约15-20元/碗
- **特点**：海南酸粉发源地，比其他地方正宗

### 英州大集（看个人兴趣）

-当地人定期举办的集市，蔬菜、水果、海鲜的露天农贸市场，每个月好多次
